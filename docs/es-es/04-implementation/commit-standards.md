# 04 - Implementación

## 4.3 Estándares de Commit

Sería beneficioso establecer estándares para cada paso de los procedimientos:

* Crear Repositorio
* Crear Issue
* Crear Branch Local
* Crear un archivo README.md fácil de entender
* Commit Local
* Push a Branch Remote
* Crear Pull Request hacia la Main Branch
* Merge Pull Request
* Cerrar Issue
