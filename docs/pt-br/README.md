# Documentation Templates

Framework de Documentação de Projetos de Software.

Template completo de documentação de software cobrindo todo o ciclo de vida, desde a proposta inicial até a operação em produção, incluindo requisitos, arquitetura, testes, deploy, governança de mudanças e entregas executivas.

Um modelo de ciclo de vida de entrega de software com comunicação entre níveis técnicos e executivos.

Cada documento contém perguntas orientadoras e estruturas prontas para facilitar o preenchimento.

## Como usar
- Clone este repositório.
- Copie os templates para seu projeto.
- Preencha os arquivos correspondentes a cada fase do ciclo de vida.
- Versione tudo junto com o código no GitHub.
